User Manual

Introduction

This website contains a simple navigation bar that allows users to move easily between different pages of the site.

How to Use the Navigation Bar
      1.	Look at the top of the page.
      2.	Click on any menu link:
            o	Home → Goes to the home page
            o	About → Displays information about the site
            o	Projects → Shows projects or work
            o	Contact → Opens the contact page

Responsive Behavior

The menu automatically adapts to:
           •	Smartphones
           •	Tablets
           •	Desktop screens

No manual configuration is needed.

Compatibility
           •	Works on modern browsers (Chrome, Edge, Firefox)
           •	Fully usable on mobile, tablet, and desktop devices

No installation or configuration is required for users.
