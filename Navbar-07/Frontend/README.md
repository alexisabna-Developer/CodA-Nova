## Overview

**Navbar-07** is a modern and fully responsive navigation bar built using **HTML5, CSS3, and Vanilla JavaScript**.

It features a professional blue user interface, responsive navigation, and a lightweight mobile menu designed for desktop, tablet, and mobile devices.

---

## Features

- Responsive navigation (mobile, tablet, and desktop)
- Professional blue UI design
- Mobile hamburger menu
- Desktop hover effects
- Login button
- Lightweight and reusable component

> ℹ️ Some navigation links are placeholders (`#` or `index.html`).

## Usage

- Open `index.html` in your browser.
- Resize the browser window to test the responsive layout.
- Click the hamburger menu on mobile devices.

---

## Behavior

| Device | Interaction |
|---------|-------------|
| Mobile | Hamburger menu toggle |
| Tablet | Responsive layout adjustment |
| Desktop | Horizontal navigation with hover effects |

---

## Installation

1. Clone or download the project.
2. Open `index.html`.
3. Make sure the CSS and JavaScript files are correctly linked:

```html
<link rel="stylesheet" href="assets/css/style.css">
<script src="assets/js/script.js"></script>
```

---

## Customization

- Edit navigation links → `index.html`
- Change styles → `assets/css/style.css`
- Modify menu behavior → `assets/js/script.js`

---

## Key Concept

Responsive navigation system:

- CSS → Layout, responsive design, and hover effects
- JavaScript → Mobile menu toggle
- `.active` class controls the mobile menu visibility
