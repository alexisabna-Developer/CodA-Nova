# User Manual - Navbar-07 | Professional Blue UI Navigation

## Introduction
CodA-Nova **Navbar-07** is a modern, lightweight, and fully responsive navigation bar.
It is built using HTML5, CSS3, and JavaScript, following a clean and professional blue UI design.

---

## Features
- Clickable CodA-Nova logo that redirects to the home page
- Navigation links:
  - Home
  - About
  - Services
  - Contact
- Login button
- Responsive navigation (mobile, tablet, and desktop)
- Mobile hamburger menu button
- Professional hover effects on desktop

> ℹ️ **Note:** Some navigation links are placeholders (`#` or `index.html`) and may not lead to real pages yet.

---

## Usage Instructions

## Usage Instructions

### Desktop
- Navigate using the horizontal navigation menu.
- Hover over navigation links to view the hover effects.
- Click the logo to return to the home page.
- Click the **Login** button to access the login page.

### Tablet
- The navigation automatically adjusts its layout for medium-sized screens.
- All navigation items remain easily accessible.

### Mobile
- Click the hamburger menu button to open the navigation menu.
- Tap the hamburger button again to close the menu.
- Tap any navigation link to navigate to the selected page.

---

## Behavior
- **Desktop:** Horizontal navigation with hover effects.
- **Tablet:** Responsive layout optimized for medium-sized screens.
- **Mobile:** Navigation menu opens and closes using the hamburger button.

---

## Responsive Behavior
- **Mobile (< 768px):** Vertical navigation with hamburger menu.
- **Tablet (768px – 991px):** Responsive layout adjustment.
- **Desktop (≥ 992px):** Horizontal navigation with hover interactions.

---

## Installation
- Open `index.html` in a web browser.
- Ensure the CSS and JavaScript files are correctly linked.
- JavaScript must be enabled for the mobile menu functionality.

---

## Customization
- Edit navigation links in `index.html`.
- Modify colors, typography, and layout in `style.css`.
- Update the mobile menu behavior in `script.js`.
