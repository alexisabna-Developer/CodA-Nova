## ðŸ“Œ Overview

A clean, mobile-first navigation menu built with pure HTML5 and CSS3.  
This template provides a simple, accessible, and responsive navigation layout without any JavaScript dependency.

> This repository represents **Navbar-01**, the first navigation template of the **CodA-Nova UI series**.
Desktop and Mobile views are currently **separated for learning and documentation purposes**.
They will be **merged into a single fully responsive navbar** in the final version.
Additional navigation patterns and layouts will be added progressively as the project evolves...
