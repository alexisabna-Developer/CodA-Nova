## Overview
**Navbar-06** is a responsive navigation bar with a **mega menu**, built using **HTML5, CSS3, and Vanilla JavaScript**.

It provides a clean layout and smooth navigation across desktop and mobile devices.

---

## Features
- Responsive navigation (mobile & desktop)
- Multi-column mega menu
- Mobile hamburger toggle
- Hover (desktop) + Click (mobile)
- Simple and clean structure

> ℹ️ Some links are placeholders (`#`)

## Technologies
- HTML5  
- CSS3 (Flexbox, Media Queries)  
- Vanilla JavaScript

## Usage
- Open `index.html` in your browser  
- Resize the screen to test responsiveness  

---

## Behavior
| Device  | Interaction |
|--------|------------|
| Mobile | Click (toggle + mega menu) |
| Desktop | Hover (mega menu) |

---

## Installation
1. Clone or download the project  
2. Open `index.html`  
3. Make sure CSS and JS are linked:

```html
<link rel="stylesheet" href="assets/css/style.css">
<script src="assets/js/script.js"></script>
```

## Customization
- Edit links → `index.html`
- Change styles → `assets/css/style.css`
- Modify behavior → `assets/js/script.js`

## Key Concept
Hybrid interaction system:

- CSS → hover (desktop)
- JavaScript → click (mobile)
- `.active` class controls visibility
