# User Manual - Navbar-06 | Responsive Mega Menu Navigation

## Introduction
CodA-Nova **Navbar-06** is a responsive navigation bar with a mega menu.  
It is built using HTML, CSS, and JavaScript.

---

## Features
- Clickable logo that redirects to the home page
- Links: Home, About, Services, Portfolio, Contact
- Mega menu with multiple columns
- Responsive navigation (mobile and desktop)
- Mobile hamburger menu button

> ℹ️ Note: Some navigation links are placeholders (`#`) and may not lead to real pages yet.
---

## Usage Instructions

### Desktop
- Mega menu opens on hover (mouse)

### Mobile
- Click the hamburger button to open the menu
- Click “Services” to open the mega menu

---
## Behavior
- Desktop: mega menu opens on hover
- Mobile: mega menu opens on click

## Responsive Behavior
- Mobile: vertical menu with toggle button
- Desktop: horizontal menu with hover mega menu

---

## Installation
- Open `index.html` in a browser
- Make sure CSS and JS files are correctly linked
- JavaScript must be enabled for menu interactions

---

## Customization
- Edit links in `index.html`
- Modify styles in `style.css`
- Change behavior in `script.js`

---

## Accessibility
- Uses semantic HTML (`<nav>`)
- Mobile-friendly interactions
- Works on touch devices
