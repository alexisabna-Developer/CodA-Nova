# Project Specifications – Navbar-06: Mega Menu

## 1. Project Title
**CodA-Nova – Navbar-06**

## 2. Project Description
Create a responsive navigation bar with a Mega Menu to organize multiple links in a structured layout.

## 3. Objectives
- Build a Mega Menu navigation  
- Keep design clean and responsive  
- Improve user experience  

## 4. Functional Requirements
- Logo linking to homepage  
- Links: Home, About, Services (Mega Menu), Portfolio, Contact  
- Mega Menu with multiple columns and titles  
- Responsive menu (desktop & mobile)  
- Sticky navbar  

## 5. Features to Code
- Mega Menu dropdown  
- Multi-column layout  
- Hamburger menu (mobile)  
- Hover effects  
- Smooth transitions  

## 6. Design Requirements
- Background: `#0C1446`  
- Text: `#ffffff`  
- Sans-serif font  
- Clean and structured layout  

## 7. Technologies Used
- HTML5  
- CSS3 (Flexbox / Grid)  
- Optional JavaScript  

## 8. Deliverables
- Responsive Mega Menu navbar  
- Clean code
